# 超长视频分段策略（>15秒）

## 核心原理

Seedance 2.0 单次生成上限 15 秒。超过 15 秒的视频采用**分段生成 + 视频延长拼接**：

1. 第 1 段正常生成（≤15秒）
2. 将第 1 段视频上传为 @视频1，用「视频延长」生成第 2 段
3. 依此类推，逐段拼接

**关键**：视频延长时，选择的生成时长 = 新增部分的时长（延长 5 秒就选 5 秒）。

超长视频的成败，不只取决于提示词长短，更取决于四件事：

1. **结构**：先建立世界，再推进变化，最后情绪收束
2. **锁定**：反复声明风格、角色、场景的一致性
3. **节奏**：整体遵守「慢 → 稍快 → 慢」
4. **衔接**：靠桥接镜头、尾帧延续和环境音消除拼接感

## 三段式叙事骨架

无论题材是剧情、广告、战斗还是科幻，超长视频都建议先拆成这三层：

### 第一层：建立世界（0-30%）

- 作用：交代空间、人物、关系、产品或世界观
- 节奏：慢
- 镜头：稳定，少切换
- 信息密度：低，优先让观众看懂

### 第二层：变化推进（30-75%）

- 作用：让画面发生明确变化
- 常见变化：情绪变化、动作升级、时间流逝、冲突展开、视觉增强
- 节奏：略快
- 镜头：可开始移动或加强动态

### 第三层：情绪收束（75-100%）

- 作用：视觉定格、氛围沉淀、意义落地
- 节奏：慢下来
- 镜头：回稳，避免突然再加大量信息

## 一致性锁定语句

长视频最容易出现风格漂移、人物变脸、场景跳变。每一段提示词都建议重复加入以下三类锁定句：

### 1. 风格锁定

```
全片保持统一视觉风格，风格不漂移，色调与质感前后一致
```

### 2. 角色锁定

```
同一角色保持一致五官、身份、服装和发型，不变脸、不换人
```

### 3. 场景锁定

```
同一场景保持一致环境、光线、背景与空间关系，不突然切换
```

> 这三句尤其适合分镜生成和视频延长场景。不要只在第一段写，后续每一段都要重复。

## 分段规则

1. 按叙事节奏切分，每段 ≤15 秒
2. 每段之间有**画面衔接点**：上段结尾状态 = 下段开始状态
3. 第 1 段正常生成，后续段用 `将@视频1延长Xs`
4. 每段标注属于整体的第几段、承接内容
5. 延长段开头优先安排 0.8-1.5 秒的微动作或环境流动，不要一上来就大幅变化

## 推荐分段方案

| 总时长 | 推荐分段 |
|--------|----------|
| 16-30秒 | 2 段（首段 15s + 延长段） |
| 31-45秒 | 3 段 |
| 46-60秒 | 4 段 |
| >60秒 | 拆分为独立场景分别生成，再用剪辑软件拼接 |

## 画面衔接方法

### 方法 1：桥接镜头法

在两个阶段之间加入 0.8-1.5 秒的自然流动镜头，优先选择低风险变化：

- 风吹动衣角、树叶、发丝、旗帜
- 光影移动、霓虹闪烁、火焰轻晃
- 云层移动、薄雾流动、尘埃漂浮
- 人物呼吸、眼神微动、手指轻动
- 环境声延续：风声、雨声、脚步声、呼吸声

适合几乎所有题材，是最稳妥的去拼接感方法。

### 方法 2：同构图推进法

不要频繁换场景，而是在同一构图中逐步推进：

- 远景 → 中景 → 近景
- 同一机位 → 光线变化 → 人物动作变化
- 同一主体位置 → 情绪或姿态逐渐变化

空间连续通常比场景跳切更自然。

### 方法 3：时间流逝法

把过渡设计成时间自然推进：

- 云移动
- 日光角度变化
- 天色由冷到暖或由明到暗
- 环境细节微变化
- 四季或天气递进

时间是最自然的过渡工具之一。

## 尾帧延续法

这是长视频拼接最稳的进阶方案，特别适合两段连续的剧情或广告动作。

### 操作方法

1. 第 1 段结尾明确写出一个可持续的定格状态
2. 第 2 段上传上一段视频，用 `将@视频1延长Xs`
3. 开头先写「接续上一段最后一帧的构图与角色状态」
4. 前 1 秒只做微动作，不要马上切大动作或换大景别

### 推荐写法

```
将@视频1延长X秒，接续上一段最后一帧的构图与角色状态，前1秒仅保留轻微呼吸、风吹发丝、环境光缓慢移动，随后再进入新的动作或情节推进
```

如果是产品广告，也可以把「轻微呼吸」替换成「反光滑动」「薄雾流动」「粒子漂浮」等更适合物体的微动。

## 音乐与声音衔接

画面接上了，不代表观感真的连贯。长视频里，声音比画面更容易暴露断裂。

### 推荐做法

1. **同一首音乐分层推进**：前段少乐器，后段逐步加入鼓点、弦乐或低频
2. **第二段提前渐入**：新段音乐可提前 0.5-1 秒渐入，避免生硬切换
3. **用环境音做桥接**：风声、雨声、脚步声、呼吸声、机械低鸣都很稳
4. **结尾减速**：最后一段让音乐和环境音一起收束，强化情绪落点

如果用户没有明确要求音乐编排，至少在提示词中写出环境音的连续性。

## 统一负面提示词

长视频建议在每一段末尾附一组统一负面限制，优先防止连续性问题：

```
禁止风格漂移，禁止角色变脸或换人，禁止突然偏色，禁止新增无关人物，禁止光线突变，禁止出现文字、字幕、LOGO、水印
```

如果是角色驱动剧情，还可补充：

```
禁止服装突变，禁止发型突变，禁止背景布局突变
```

## 分镜时间分配策略（短剧/剧情向补充）

> 以下内容补充短剧和剧情类视频的时间分配方法论，与上方"三段式叙事骨架"配合使用。核心原则：分镜时间分配的目标不是把剧情平均切开，而是让观众在有限时长内完成"看懂主体、进入情绪、接收信息、获得变化"四件事。

### 台词与动作估时公式

单镜头估时先看是否有台词；没有台词时再按动作、情绪、空镜或特效估时。

| 镜头类型 | 公式/时长 | 使用说明 |
|---|---:|---|
| 有台词（正常语速） | `round(字数÷8)+1秒` | 普通对白、解释、问答 |
| 有台词（争吵/急促） | `round(字数÷10)+1秒` | 语速快、打断、情绪激烈 |
| 有台词（深情/遗言） | `round(字数÷5)+1秒` | 慢速表达、停顿多、情绪重 |
| VO 旁白 | `round(字数÷7)+1秒` | 旁白、内心独白、说明性文字 |
| 微动作/常规动作 | `2秒` | 回头、抬眼、递物、坐下、轻触 |
| 复杂连续动作 | `4秒` | 起身走到门口、翻找物件、连续闪避 |
| 强烈情绪反应 | `3秒` | 崩溃、震惊、哭泣、愤怒爆发 |
| 环境/空镜/静物 | `2秒` | 场景交代、关键物特写、氛围缓冲 |
| 慢动作特效 | `3-4秒` | 升格、水花凝固、碎片飞散、时间停滞 |

**优先冲突原则**：镜头同时含台词和动作时，取两者计算结果中的**较大值**。

示例：

```text
台词 18 字，正常语速：round(18÷8)+1 = 3 秒。
同镜还有复杂连续动作：4 秒。
最终该镜按 4 秒估时。
```

**时长边界**：
- 每个镜头严格限制在 `12 秒` 以内
- 若单镜估时超过 12 秒，应拆成两个镜头，或删减台词/动作

### 基础镜头时长参考

| 镜头类型 | 建议时长 | 分配要点 |
|---|---:|---|
| 信息建立镜头 | 1.5-3 秒 | 画面要清楚，不要同时塞太多主体 |
| 人物反应镜头 | 1.5-3 秒 | 微表情需要 1 秒以上停留 |
| 关键动作镜头 | 2-4 秒 | 动作要有起点和落点 |
| 对话/口型镜头 | 2-5 秒 | 一镜一句话最稳 |
| 转场镜头 | 0.5-2 秒 | 只负责连接，不额外承担复杂剧情 |
| 氛围镜头 | 1.5-4 秒 | 用于缓冲节奏和建立情绪 |
| 反转揭示镜头 | 2-4 秒 | 揭示后要给观众看清楚 |
| 高强度动作镜头 | 1-2.5 秒 | 可以短，但必须有明确视觉中心 |
| 慢动作/升格镜头 | 2-5 秒 | 慢动作太短会失去冲击力 |
| 结尾钩子镜头 | 2-4 秒 | 结尾需要"信息变化"，不是只停在漂亮画面 |

### 按剧情功能分配时间

> 以下占比是六类镜头功能的时长分配，可与上方三段式叙事百分比（建立世界 0-30% / 变化推进 30-75% / 情绪收束 75-100%）叠加使用，两者不冲突——前者是叙事三段节奏，后者是镜头功能时长配比。

| 剧情功能 | 推荐占比 | 说明 |
|---|---:|---|
| 开场钩子 | 10%-20% | 短剧要快进冲突，不能铺太久 |
| 场景/关系建立 | 10%-20% | 只交代必要信息 |
| 动作推进 | 25%-40% | 主体必须持续做事 |
| 情绪反应 | 10%-20% | 让观众理解人物心理 |
| 揭示/反转 | 10%-20% | 给明确的信息变化 |
| 尾钩 | 5%-10% | 留问题，不解释过度 |

### 按情绪类型分配时间

| 情绪类型 | 时间策略 | 镜头节奏 |
|---|---|---|
| 紧张悬疑 | 前快后停 | 前半段快速给异常，后半段停在反应和揭示 |
| 悲伤压抑 | 慢进入、长停顿 | 多给面部、手部、空镜，动作少但停留久 |
| 动作追逐 | 快切 + 少量定点 | 每镜 1-3 秒，关键撞击或闪避可升格 |
| 浪漫心动 | 留白和前景虚化 | 镜头较慢，反应镜头 2 秒以上 |
| 恐怖惊吓 | 安静铺垫 + 突然变化 | 70% 铺垫，30% 爆点和反应 |
| 反转爽点 | 误导 + 停顿 + 揭示 | 揭示前必须留 1-2 秒心理停顿 |

### 按镜头数量分配

| 总时长 | 推荐镜头数 | 单镜平均 | 说明 |
|---|---:|---:|---|
| 5 秒 | 1-2 镜 | 2.5-5 秒 | 只做一个动作或一个钩子 |
| 10 秒 | 2-4 镜 | 2.5-5 秒 | 一个微型冲突 |
| 15 秒 | 3-5 镜 | 3-5 秒 | 一个完整短剧片段 |
| 30 秒 | 6-10 镜 | 3-5 秒 | 一个完整小场景 |
| 60 秒 | 10-18 镜 | 3-6 秒 | 一个完整短剧场景 |
| 90 秒 | 15-25 镜 | 3.5-6 秒 | 一个场景组或连续剧情段 |

**过密风险**：15 秒超过 7 镜 AI 容易主体漂移；30 秒超过 12 镜容易信息堆叠；60 秒超过 20 镜一致性下降。

**过松风险**：15 秒只有 1 镜容易只有氛围没有剧情；30 秒少于 4 镜容易节奏拖；60 秒少于 8 镜短剧信息密度不足。

### 结尾设计原则

结尾必须有"信息变化"，不是只停在漂亮画面：

- **揭示**：证据、身份、真相浮出
- **反转**：颠覆观众预期
- **危险逼近**：威胁升级
- **悬念收束**：留下下一段问题

结尾钩子镜头 2-4 秒，揭示后要给观众看清楚。**不要在最后 2 秒引入复杂新设定**。

### 各时长分镜模板

**15 秒分镜模板**：

```text
0-3秒：开场钩子，直接给异常/冲突/危险。
3-6秒：交代主体、场景、关键物。
6-10秒：主体行动，冲突升级。
10-12秒：人物反应，表情停顿。
12-15秒：揭示或尾钩。
```

15 秒四种子类型：

| 类型 | 时间分配 | 适合内容 |
|---|---|---|
| 悬疑钩子型 | 3 + 3 + 4 + 2 + 3 | 异常出现、证据揭示、人物震惊 |
| 情绪爆发型 | 2 + 4 + 4 + 3 + 2 | 争吵、崩溃、隐忍、决裂 |
| 动作追逐型 | 2 + 3 + 5 + 2 + 3 | 开跑、追逐、闪避、发现出口 |
| 反转揭示型 | 3 + 4 + 3 + 2 + 3 | 误导、线索、停顿、真相 |

**30 秒分镜模板**：

```text
0-4秒：强钩子。
4-8秒：场景和人物关系。
8-14秒：第一行动。
14-20秒：冲突升级。
20-24秒：情绪停顿。
24-28秒：揭示/反转。
28-30秒：尾钩。
```

**60 秒分镜模板**：

```text
0-5秒：强钩子。
5-12秒：背景补足。
12-18秒：目标出现。
18-28秒：冲突推进 1。
28-38秒：冲突推进 2。
38-45秒：情绪低点。
45-53秒：真相揭示。
53-60秒：结尾钩子。
```

> 60 秒不是 4 个 15 秒拼接，必须有主冲突线。每 10-15 秒需要一次信息变化。第 45 秒后不要再引入新的复杂设定，只做揭示或钩子。

### 台词镜头拆分规则

| 问题 | 处理方式 |
|---|---|
| 一句台词超过 12 秒 | 拆成两镜，前半句给说话人，后半句给听者反应 |
| 同镜既有长台词又有复杂动作 | 拆成"动作镜头 + 台词反应镜头" |
| 争吵台词太密 | 每句短台词独立成镜，或用快速正反打 |
| 深情台词太长 | 保留关键句，增加表情停顿，不要塞满全句 |
| VO 旁白覆盖多个画面 | 按 VO 总时长分配多个画面，不要求每个画面都有口型 |

### 分镜时间分配检查清单

- [ ] 开场 3 秒内是否有明确钩子
- [ ] 每个镜头是否只有一个核心动作或信息变化
- [ ] 关键动作是否至少有 2 秒
- [ ] 重要表情是否至少停留 1.5 秒
- [ ] 台词镜头是否按语速公式估时
- [ ] 同镜包含台词和动作时，是否按较大值取时长
- [ ] 单镜是否严格控制在 12 秒以内
- [ ] 转场是否预留 0.5-1 秒
- [ ] 结尾是否有明确的信息变化，而不是只停在氛围画面
- [ ] 是否避免在最后 2 秒引入复杂新设定
- [ ] 是否每 10-15 秒出现一次冲突升级、线索揭示或情绪变化

## 分镜创作六步法（短剧/剧情向补充）

> 以下方法论补充"三段式叙事骨架"的设计层面——在写分段提示词之前，先用这六步规划镜头叙事。
>
> 💡 **建议阅读顺序**：先读本六步法（设计层面：镜头怎么设计），再查上方"分镜时间分配策略"（量化层面：每个镜头给多少秒），最后结合"三段式叙事骨架"（整体节奏排布）。三者构成"整体→单镜设计→单镜时长"的递进。

### 第一步：确定跟随对象

明确镜头主要跟谁走。短剧中通常有明确的主角，但群体场景需要决定焦点人物。

### 第二步：设计运动路线

规划主体在空间中的移动路径——从哪里开始、经过哪些位置、最终停在哪里。

### 第三步：用镜头运动解释空间

通过推、拉、升、降、跟拍展示环境。不要只写"镜头移动"，要写镜头**为什么**这样移动：
- 推入 = 聚焦 / 压迫 / 亲密
- 拉远 = 揭示 / 孤独 / 疏离
- 升起 = 升华 / 全知 / 逃离
- 下降 = 压抑 / 落地 / 聚焦细节
- 跟拍 = 同步 / 陪伴 / 追逐

### 第四步：安排主体关系变化

控制人物、车辆、物体之间的距离和交互。关系变化是短剧叙事的核心：
- 两人从远到近 = 关系拉近 / 威胁逼近
- 两人从近到远 = 疏离 / 逃离 / 决裂
- 一人静止一人移动 = 权力不对等 / 主动与被动

### 第五步：设计停顿点

为叙事制造节奏和悬念。停顿点不是空镜，而是**情绪锚点**：
- 动作后的表情停留（给观众时间理解情绪）
- 揭示前的悬念停顿（制造预期）
- 沉默的对峙（积累张力）
- 环境氛围缓冲（节奏换气）

### 第六步：明确结尾的信息变化

用爆发、转折、揭示或释放完成收束。结尾不是"画面结束"，而是**信息变化**：
- 揭示：证据、身份、真相浮出
- 反转：颠覆观众预期
- 危险逼近：威胁升级
- 悬念收束：留下下一段问题

> 这六步与上方"三段式叙事骨架"（建立世界→变化推进→情绪收束）配合使用：六步法解决"每个镜头怎么设计"，三段式解决"整体节奏怎么排布"。
